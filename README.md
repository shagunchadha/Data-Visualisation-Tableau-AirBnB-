# Airbnb Data Analysis Dashboard (Tableau)

## Project Overview
This project involves analysing a real-world Airbnb dataset using Tableau Public.
The dashboard was built by following Alex The Analyst's Tableau Full Project tutorial series.

## Dataset
- File: AirBnB Full Project.twbx
- Source:Airbnb public listings data
- Kaggle DataSet Link: https://www.kaggle.com/datasets/alexanderfreberg/airbnb-listings-2016-dataset
- Tool Used: Tableau Public

## What the Dashboard Shows

**Price by Location / Zip Code**
- Geographic breakdown of average listing prices across zip codes

**Revenue Trends by Year**
- How total Airbnb revenue has changed over time

**Bedrooms vs. Price**
- How the number of bedrooms impacts average listing price

**Availability Calendar**
- Seasonal availability patterns across listings
